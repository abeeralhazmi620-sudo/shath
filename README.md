Interactive Arabic Lesson: The Instrument Noun (Ism Al-Ala)

An engaging, interactive Single Page Application (SPA) designed for 3rd-grade middle school students (Saudi Curriculum) to explore the grammatical concept of "The Instrument Noun" (اسم الآلة).

This project transforms a traditional grammar lesson into a dynamic digital experience, moving from concrete examples to abstract rules through interactive storytelling and data visualization.

🌟 Key Features

Step-by-Step Pedagogical Journey: The application follows a logical 5-stage learning path:

Surprise Box: An interactive "reveal" mechanic to spark curiosity about daily tools.

Interactive Story: A narrative about a carpenter where students can hover over tools to see their root verbs.

Concept Flip-Cards: A "Click-to-Swap" feature that hides the action (verb) and shows the tool (noun) to reinforce the relationship between the two.

Data Visualization: A dynamic Chart.js doughnut chart that categorizes instruments by their usage environment (Home, Work, School).

Lesson Foundation: A concluding section that prepares students for the formal weights (Awzan) of the nouns.

Fully Responsive UI: Optimized for desktops, tablets, and mobile devices using Tailwind CSS.

RTL Support: Native Right-to-Left layout designed specifically for the Arabic language.

🛠️ Technologies Used

Frontend: HTML5, CSS3, JavaScript (Vanilla ES6+).

Styling: Tailwind CSS (via CDN).

Charts: Chart.js for interactive data representation.

Typography: Google Fonts (Tajawal font family).

🚀 How to Run

Since this is a standalone SPA, no installation is required:

Download the lesson_interactive.html file.

Open the file in any modern web browser (Chrome, Firefox, Safari, or Edge).

📁 Project Structure

lesson_interactive.html: The main entry point containing all HTML structure, Tailwind styles, and JavaScript logic.

State Management: Managed via simple Vanilla JS variables and DOM manipulation for smooth transitions between lesson stages.
